### 20/06/14 ABC170

E/Fに少しずつチャレンジできているのは成長を感じる
Dまでの精度と速度をもっとあげたい

#### 結果

|A|B|C|D|E|F|
|:--:|:--:|:--:|:--:|:--:|:--:|
|AC|AC|AC|AC+3WA||timeup|
|0'55"|3'50"|7'46"|66'09"|||

Fいけそう、と思いきやロジック的にダメな部分があってちょっと厳しかった。
Dにこんなにかかったのはダメだったなあ
$\displaystyle{\sum^N_{k=1}} \dfrac{1}{k} = \log N$だから、素直に$N\log N$でいけたのになあ

#### 復習したい

- F(BFSの枝刈り)
  - BFSの1step(dequeに入れる1要素)あたりkマス進みうるから$O(HWK)$だった。
  - 到達済みの場所を持っておくことで何度も訪ねなくていいので$O(HW)$に落ちると考えて、その方針自体はOKだった（ここまではコンテスト中にできた）
  - 同じ手数で到達できる場所に対しては到達済みでもそれより先に訪問できるので、それの考慮が必要だった。
